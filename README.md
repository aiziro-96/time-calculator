# ⏱ 時間計算アプリ

## 🔗 デモ

👉 https://aiziro-96.github.io/time-calculator/

---

## 📝 概要

分や時間を入力すると、自動で時間に変換・計算できるWebアプリです。
シンプルで直感的に使えることを意識して作成しました。

---

## ✨ 主な機能

* 分の足し算（例：90+30）
* 時間形式の計算（例：1:52 + 45）
* 小数形式の入力（例：1.52）
* 日本語入力対応（例：1時間52分）
* 自動計算
* 履歴保存機能
* 個別削除 / 全削除（確認ダイアログ付き）
* コピー機能
